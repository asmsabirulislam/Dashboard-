⚡ QUICK SETUP (5 MINUTES)

1. INSTALL PYTHON (if not already installed)
   - Windows: https://python.org (check "Add Python to PATH")
   - macOS: brew install python3
   - Linux: sudo apt-get install python3 python3-pip

2. INSTALL DEPENDENCIES
   Windows:
   │ cd path\to\dashboard
   │ python -m pip install -r requirements.txt
   
   macOS/Linux:
   │ cd path/to/dashboard
   │ python3 -m pip install -r requirements.txt

3. RUN DASHBOARD
   Windows:
   │ streamlit run powerbi_enterprise_dashboard.py
   
   macOS/Linux:
   │ streamlit run powerbi_enterprise_dashboard.py

4. ACCESS DASHBOARD
   - Opens automatically at http://localhost:8501
   - Or copy URL from terminal

🔧 TROUBLESHOOTING

ISSUE: "Python not found"
SOLUTION:
   Windows: Add to PATH or use full path
   Linux/Mac: Use python3 instead of python

ISSUE: "Module not found"
SOLUTION:
   pip install -r requirements.txt --upgrade --force-reinstall

ISSUE: "Port already in use"
SOLUTION:
   streamlit run powerbi_enterprise_dashboard.py --server.port 8502

ISSUE: Dashboard won't load
SOLUTION:
   streamlit cache clear
   streamlit run powerbi_enterprise_dashboard.py

🎓 CUSTOMIZATION

To use your own data:

1. Prepare a CSV or Excel file with columns:
   - Date (datetime)
   - Bank (string)
   - Firm (string)
   - Party (string)
   - Buyer (string)
   - Sales Person (string)
   - Invoice Value (numeric)
   - Status (Accepted/Pending/Rejected)
   - Payment Status (Received/Pending/Rejected)

2. Find this line in the code:
   @st.cache_data
   def generate_sample_data():

3. Replace with your data loading code:
   @st.cache_data
   def generate_sample_data():
       return pd.read_excel('your_file.xlsx')

4. Save and refresh the browser
